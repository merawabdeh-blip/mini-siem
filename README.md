# 🚀 Mini SIEM System

## 📌 Overview
This project is a **Mini SIEM (Security Information and Event Management)** system that performs:

- Real-time log monitoring
- AI-based anomaly detection
- Rule-based attack detection
- Alert generation and visualization

The system is designed for educational and experimental cybersecurity purposes.

---

## 🧠 Features

- 📥 Log ingestion (API)
- 🔄 Log normalization
- 🧠 Hybrid detection engine (AI + Rules)
- 🚨 Alert generation
- 📊 Dashboard (basic)
- 🤖 Isolation Forest model for anomaly detection

---

## 🏗️ System Architecture

Log Sources → Preprocessing → AI Engine → Alert Generator → Dashboard
